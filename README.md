# Contact Management System

A simple Contact Management System built using PHP and MySQL. This project allows users to manage contact details through basic CRUD operations.

## Features

- Add New Contact
- View Contact List
- Edit Contact Details
- Delete Contact
- Search Contact by Name

## Technologies Used

- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Project Structure

```
Contact-Management/
│── config.php
│── index.php
│── add_contact.php
│── edit_contact.php
│── update_contact.php
│── delete_contact.php
│── search.php
│── style.css
```

## Database

Database Name:

```
contact_management
```

Table Name:

```
contacts
```

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into `xampp/htdocs/`.
4. Create the `contact_management` database in phpMyAdmin.
5. Import the SQL file (if available).
6. Open your browser and visit:

```
http://localhost/Contact-Management/
```
